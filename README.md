# Boilerplates
Standart boilerplate for Webpack configs.
Includes: 
- webpack
- webpack-dev-server + Hot Module Replacement
- @babel/preset-env + @babel/plugin-transform-runtime" + "@babel/plugin-proposal-class-properties"
- file-loader
- html-webpack-plugin
- SASS-loader
- separate configs for DEV/PROD
